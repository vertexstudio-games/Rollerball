🎮 Unity Learning Project
📌 About the Project

This project was developed as part of a Unity learning course. It focuses on fundamental game development concepts and serves as a practical introduction to building games using Unity and C#.

🚀 Features

- Player movement
- Physics-based collision detection
- Trigger-based interactions
- Game over system
- Basic UI implementation
-Score system
-Item collection

🛠️ Technologies Used

- Unity
- C#

📚 What I Learned

- Fundamentals of game development with Unity
- Player input and movement mechanics
- Difference between collision and trigger systems
- Basic UI creation and management
- Game logic and scripting structure
- Enemy behavior basics (if applicable, you can remove if not used)

🎯 Purpose

This project was created for learning purposes, with the goal of building a solid foundation in Unity development and game programming.
